Python

Python Hakkında

Python, öğrenmesi kolay ve çok yönlü bir programlama dilidir. Veri bilimi, yapay zeka ve web geliştirme gibi alanlarda yaygın olarak kullanılır.

  • Kolay okunabilir sözdizimi
  • Zengin kütüphane desteği
  • Geniş kullanım alanı

Python Sözdizimi Örneği:


# Basit bir Python fonksiyonu
def selamla(isim):
    return f"Merhaba {isim}!"

# Liste işlemleri
sayilar = [1, 2, 3, 4, 5]
kareler = [x**2 for x in sayilar]

# Sınıf tanımlama
class Kullanici:
    def __init__(self, isim, yas):
        self.isim = isim
        self.yas = yas

Ada

Ada Hakkında

Ada, güvenilirlik ve güvenlik odaklı, askeri ve havacılık uygulamaları için geliştirilmiş bir dildir.

  • Güçlü tip kontrolü
  • Paralel programlama desteği
  • Yüksek güvenilirlik

Ada Sözdizimi Örneği:


with Ada.Text_IO; use Ada.Text_IO;
procedure Hello is
   Name : String(1..30);
   Last : Natural;
begin
   Put_Line("İsminizi girin:");
   Get_Line(Name, Last);
   Put_Line("Merhaba " & Name(1..Last) & "!");
end Hello;

Fortran

Fortran Hakkında

Fortran, bilimsel hesaplamalar için geliştirilmiş ilk yüksek seviyeli programlama dilidir.

  • Bilimsel hesaplama optimizasyonu
  • Matris işlemleri desteği
  • Yüksek performans

Fortran Sözdizimi Örneği:


program hello
    implicit none
    character(len=30) :: name
    
    print *, "İsminizi girin:"
    read *, name
    print *, "Merhaba ", trim(name), "!"
end program hello

program HelloWorld;

  SysUtils;

  name: string;

  ReadLn;
